# Amazone clone project #
A responsive front-end clone of the Amazon homepage built using HTML5 and CSS3. This project recreates the look and feel of Amazon's landing page, including the navigation bar, hero section, product categories, and footer. It was developed to strengthen my frontend web development skills.
## project overview ##
This project is a static clone of the Amazon homepage created using only HTML and CSS. It focuses on building a clean, structured, and visually appealing user interface while following modern web design practices.
# Features #
1.Amazon-inspired navigation bar  
2.Search bar with category selection  
3.Delivery location section  
4.Hero banner  
5.Product category cards  
6.Responsive page layout  
7.Footer with multiple sections  
8.Font Awesome icons  
# Technologies used #
1.HTML5  
2.CSS3  
3.Font Awesome  
4.Visual Studio Code
